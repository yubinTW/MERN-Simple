## Get Start

## Depenency

- node
- docker (for run a mongo container)

### Clone this project

```
git clone https://github.com/yubinTW/MERN-Simple.git
cd MERN-Simple
```

### Run mongo with docker container

```
docker run -d -p 27017:27017 mongo
```

### Start frontend project

```
cd frontend
npm i
npm run start
```

### Start backend project

```
cd backend
npm i
npm run build
npm run start
```
